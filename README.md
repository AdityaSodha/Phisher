# Installation 
## For linux(Debian-based)
```shell
apt update
apt upgrade
apt install git
apt install python3
git clone https://github.com/AdityaSodha/Phisher
cd Phisher
python3 Phisher
```
## For Termux
```shell
apt update
apt upgrade
pkg install python3
pkg install git
git clone https://github.com/AdityaSodha/Phisher
cd Phisher
python3 Phisher
```
